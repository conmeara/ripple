1. What it does — Ripple is a desktop tool for creating short motion graphics through plain-English prompts instead of manual After Effects work. It builds on 1Code’s chat-based agent workflow and adapts it for HyperFrames. The goal is to give motion designers an agentic workflow without asking them to learn GitHub, coding tools, dependency installs, or project setup. Users ask an agent to change text, timing, layout, and animation, preview quickly, and export an MP4.

2. Goals & success criteria — Success means a user can open a project, describe a title card or lower third, iterate through chat or frame-anchored comments, and export a usable 1080p video in one session. "Done" is reliable HyperFrames editing, fast preview updates, and professional-looking output without hand-coding.

3. Scope & constraints — In scope for MVP: HyperFrames preview/rendering, HTML/CSS/GSAP animations, MP4 export, asset import, and isolated worktrees for comment-driven revisions. Ripple will build on 1Code, reusing its chat flow and support for existing Claude Code or Codex access. The app should hide Git, worktrees, dependencies, and installs. Out of scope: cloud sync, multi-user collaboration, mobile, and full After Effects export. Constraints: HyperFrames requires Node.js 22+ and FFmpeg for local rendering, and the agent must stay inside the project boundary.

4. Key roles & users — Primary users are motion designers, editors, marketers, indie founders, and agency teams creating social promos, title cards, lower thirds, and explainers. A key persona wants AI help with motion graphics but not a developer workflow.

5. Domain context — This sits between motion design tools and AI coding-agent tools. HyperFrames uses HTML compositions, timed clips, GSAP timelines, and deterministic local rendering, so the agent must generate structured video markup rather than generic app code. Ripple’s main UX addition is a Frame.io-style commenting layer so motion designers can leave feedback in a familiar surface while the app handles the technical plumbing.
