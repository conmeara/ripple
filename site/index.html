Why Ripple

The app handles the parts motion teams should never have to think about.

Agentic editing is powerful, but the raw workflow asks creative people to manage developer tools. Ripple turns that into a motion app: projects, preview, comments, revisions, and export.

01

Download and start

Ripple bundles the local app workflow so designers do not have to install skills, wire dependencies, or understand GitHub before making a video.

02

Connect your agent

Use the Claude or Codex access you already have. Ripple keeps the provider details out of the creative surface.

03

Edit in the preview

Open a project, leave frame comments, compare proposed versions, and accept the changes that make the cut.

01

Made for motion designers, editors, and video creators

Ripple is for people who want agentic video editing without learning repo setup, terminal commands, worktrees, package installs, or agent runtime plumbing.

02

Comments become isolated agent sessions

Each frame or range note can run on its own proposed version. Click between comments, compare Main and Proposed, and let Ripple handle the merge path.

03

Visual context makes agents better editors

Ripple gives agents current frames, frame sheets, timing, and project context quickly, so they can reason about the composition they are changing.
