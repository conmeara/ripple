# Ripple Logo Export Candidate - Black And White

Generated as a monochrome follow-up to `build/ripple-logo-export`.

These files are not wired into the app yet. They mirror the names currently
expected by the Electron packaging config so they can be copied into `build/`
when the logo is ready to implement.

- `icon.png`: 1024x1024 black-and-white source app icon for Linux and notification fallback.
- `icon.icns`: macOS app icon bundle.
- `icon.ico`: Windows app, installer, and uninstaller icon.
- `icon.iconset/`: macOS source slices used to build `icon.icns`.
- `trayTemplate.svg`: monochrome template-style tray/menu glyph source.
- `trayTemplate.png`: 22x22 template raster.
- `trayTemplate@2x.png`: 44x44 Retina template raster.
- `preview-64.png`: quick small-size preview.
- `preview-22.png`: quick tiny-size preview.
- `source.svg`: editable color SVG source for this candidate.
